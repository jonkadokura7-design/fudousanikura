# 不動産いくら？ (Real Estate How Much?)

Luxury Real Estate Price Checker Web App.
AI-powered interface for browsing real estate transaction prices in Japan.

## Features
- **Luxury UI**: Glassmorphism design with Gold/Dark theme.
- **Easy Search**: Select Prefecture and City to find prices instantly.
- **MLIT API**: Uses official Ministry of Land, Infrastructure, Transport and Tourism data.
- **Demo Mode**: Includes fallback mock data if API limits or CORS issues occur.

## How to Use
1. Open `index.html` in your web browser.
2. Select a Prefecture (e.g., Tokyo).
3. Select a City/Ward (e.g., Minato-ku).
4. Click **"調べる"** (Search).

## Deployment
This project is built with vanilla HTML/CSS/JS (React via CDN).
It can be deployed directly to **GitHub Pages** or any static hosting.
